# Operations Levels

A way to describe how mature a business's operations are. Used during discovery and proposal scoping.

## Level 1 — Improvised

- Work coordinated via chat, email, memory.
- Spreadsheets are the system of record.
- New hires shadow existing staff to learn.
- Owner is involved in most operational decisions.

## Level 2 — Tooled

- Off-the-shelf tools in use (booking system, CRM, accounting).
- Tools are not deeply integrated.
- Manual handoffs between tools.
- Some written procedures.

## Level 3 — Connected

- Tools are integrated where it matters.
- Data flows automatically between core systems.
- Reports come from real data, not memory.
- New hires onboard with documentation, not osmosis.

## Level 4 — Optimized

- Operations are measured and improved continuously.
- Automation handles repetitive work.
- Exceptions are visible and managed.
- AI assistance is used in defined, governed ways.

## How we use this

- Identify the client's current level.
- Identify the next level worth aiming for.
- Scope the project to move them one level forward — not four.
