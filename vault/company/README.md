# vault/company

The **single source of truth** about Psilodigital itself. Anything an agent needs to claim about who we are, what we sell, how we price, who we serve — it lives here.

## Suggested files (create as needed)

- `about.md` — origin story, mission, team
- `icp.md` — ideal client profile(s)
- `services.md` — what we sell (custom software, automation, PsiloBooking)
- `voice.md` — brand voice and tone guide
- `rate-card.md` — day rates, package pricing, retainer tiers (Internal)
- `case-studies.md` — real prior work (with permission)
- `legal.md` — entity info, registered address, standard terms
- `partners.md` — referral partners, tech partners

## Rules

- **Treat as authoritative.** Agents should cite this folder before making any claim about Psilodigital.
- **Versioned.** Every change is a commit. PR or commit message should explain why.
- **No external sharing of `rate-card.md` or `legal.md` without operator approval.**
