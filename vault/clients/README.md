# Clients

One folder per client.

Each client folder follows the structure in `_template/`:

- `context.md` — facts, assumptions, open questions
- `meetings.md` — meeting notes and briefs
- `opportunities.md` — possible work, ranked
- `proposals.md` — proposal drafts and history

Use kebab-case slugs (`acme-tours`, `boats-of-lisbon`).

Do not mix client contexts.
Do not put real secrets here.
