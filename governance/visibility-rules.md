# Visibility Rules

## Always allowed

Agents may read:
- vault/company/
- vault/products/
- vault/templates/
- vault/references/
- relevant vault/clients/[client]/
- relevant skills/
- relevant agents/
- relevant workflows/

## Ask before reading

Agents should ask before reading:
- financial records
- legal documents
- sensitive client data
- archived client material
- personal notes

## Never read by default

Agents must not read:
- real secrets
- private personal notes
- unrelated client folders
- logs containing sensitive data
