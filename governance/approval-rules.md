# Approval Rules

## No approval needed

- Drafting internal notes
- Preparing meeting briefs
- Creating proposal drafts
- Creating task breakdowns
- Summarizing provided content

## Approval required

- Sending emails
- Updating client-facing documents
- Creating GitHub issues
- Changing production systems
- Calling live APIs
- Creating invoices
- Modifying client data

## Always forbidden without explicit human instruction

- Deleting production data
- Sending contracts
- Sending invoices
- Sharing secrets
- Contacting clients autonomously
