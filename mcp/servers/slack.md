# Slack MCP Server

## Purpose

Allow agents to read approved Slack channels and draft messages for internal team coordination.

## Possible use cases

- Surface unread mentions and action items
- Summarize a busy channel into a brief
- Draft a weekly internal update
- Detect blockers raised in chat

## Permissions

Start read-only.

Write actions require human approval:
- posting messages
- sending DMs
- creating channels
- adding members

DMs are never sent automatically.
