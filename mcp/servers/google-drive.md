# Google Drive MCP Server

## Purpose

Allow agents to read documents, sheets, and files shared by clients or internal teams.

## Possible use cases

- Pull client-shared documents into discovery prep
- Read internal templates and references
- Summarize a folder of materials
- Locate the latest version of a proposal or scope

## Permissions

Start read-only.

Write actions require human approval:
- creating files
- editing files
- changing sharing settings
- moving or deleting items

Never modify client-shared files without explicit approval.
