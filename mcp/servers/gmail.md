# Gmail MCP Server

## Purpose

Allow agents to read and draft email related to client work.

## Possible use cases

- Pull threads relevant to a specific client
- Draft follow-up emails
- Summarize long threads into meeting prep
- Detect unanswered client messages

## Permissions

Start read-only.

Write actions require human approval:
- creating drafts
- sending email
- modifying labels
- archiving messages

Sending email is never automatic.
