# Client Meeting Prep Workflow

## Purpose

Prepare Psilodigital for a client discovery meeting.

## Trigger

Use when a meeting is scheduled or when preparing for an exploratory call.

## Inputs

- Client name
- Website
- Known notes
- Meeting goal

## Primary agent

- business-analyst

## Supporting agents

- sales-strategist

## Skills used

- discovery-call-prep

## Vault context used

- vault/company/
- vault/services.md
- vault/clients/[client]/
- vault/references/common-sme-bottlenecks.md

## Procedure

1. Read client context.
2. Read Psilodigital company/services context.
3. Apply discovery-call-prep.
4. Produce meeting brief.
5. List assumptions and open questions.

## Output

Output path:
`outputs/[client]/meeting-brief.md`

## Approval points

No approval required for internal meeting prep.
