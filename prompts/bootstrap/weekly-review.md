# Bootstrap: Weekly Review

Use this prompt every Monday (or whatever day works) to run the WBR.

---

Read `SYSTEM.md` and `AGENTS.md`. Then run the `weekly-business-review` workflow for the last 7 days.

Cover, in this order:

1. **Pipeline** — new leads, advanced deals, lost deals, top 3 active with next action.
2. **Delivery** — per active client: green/yellow/red and one-line reason.
3. **PsiloBooking** — headline metrics, incidents, open P0/P1.
4. **Finance** — MRR/ARR snapshot, outstanding invoices, runway note.
5. **Open decisions** — anything across the business waiting on a person.

Save to `vault/company/wbr/<YYYY-MM-DD>.md`. Then draft an internal Slack post and **wait for my approval** before posting.

Keep it to one screen. If something is critical, lead with it.
