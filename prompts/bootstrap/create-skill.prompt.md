# Create Skill Prompt

Create a new skill folder under `skills/[skill-name]`.

The skill must include:
- SKILL.md
- templates/
- references/
- examples/

The SKILL.md must include:
- YAML metadata
- when to use
- inputs
- procedure
- output format
- rules
- references
- templates

The skill should be specific, reusable, and testable.
