# Create Client Context Prompt

Create a new client folder under `vault/clients/[slug]/`.

The folder must include:
- context.md
- meetings.md
- opportunities.md
- proposals.md

Copy the structure from `vault/clients/_template/`.

The context.md must include:
- basic information
- business model
- current situation
- pain points
- opportunities
- facts
- assumptions
- open questions

Rules:
- Use kebab-case for the slug.
- Do not invent facts.
- Mark assumptions clearly.
- Do not mix client contexts.
