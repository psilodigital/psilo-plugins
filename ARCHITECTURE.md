# Architecture Overview

The Psilodigital Business Agent System is organized into seven layers:

1. Control Layer — global rules and tool instructions
2. Knowledge Layer — company, product, client, and reference knowledge
3. Capability Layer — reusable skills
4. Role Layer — agent role definitions
5. Workflow Layer — business processes combining agents and skills
6. Tool Access Layer — future MCP/connectors/env configuration
7. Execution Layer — scripts, prompts, outputs, logs, and future admin UI

The goal is to make Psilodigital's work repeatable, portable, and AI-operable.
